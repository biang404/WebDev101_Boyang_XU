01 hello world typescript

📝 Question :

Comparez le fichier hello.js et hello.ts et déduisez l'étape de transpilation. À quoi sert le typage fort dans TypeScript qui n'existe pas dans JavaScript ? Répondez dans un fichier texte au format markdown.

✅ Réponse :

La comparaison entre le fichier hello.js et hello.ts

En TypeScript, nous devons spécifier le type des variables (par exemple, dans hello.ts, le type de message2 est string), alors qu'en JavaScript, ce n'est pas nécessaire. De plus, pour définir une variable constante, les mots-clés utilisés en TypeScript et en JavaScript sont également différents.

L'étape de transpilation :

Le fichier hello.ts est écrit en TypeScript, un langage qui ajoute des fonctionnalités supplémentaires à JavaScript, notamment le typage statique. Lors de la transpilation (conversion du code TypeScript en JavaScript), le compilateur TypeScript (tsc) supprime les annotations de type et génère un fichier JavaScript standard.

Exemple :

const message2: string = "Hello World";

devient :

var message1 = "Hello World";
Rôle du typage fort dans TypeScript :
  • ✅ Détection des erreurs de type avant exécution.
  • ✅ Meilleure lisibilité et compréhension du code.
  • ✅ Autocomplétion plus précise dans les IDEs.
  • ✅ Moins de bugs liés aux types, meilleur maintien du code.
  • ✅ Support des concepts avancés comme les interfaces, génériques, SOLID.

En résumé : le typage fort dans TypeScript ajoute une couche de sécurité et de clarté qui n'existe pas dans JavaScript seul.

02 hello astro typescript

📝 Question :

Créez une application web servie avec Astro, en TypeScript, qui affiche "Bonjour, TypeScript!" dans la page web :

  • Créer programmatiquement (depuis le script TypeScript) un <div> avec un id "app"
  • Créer un <h1> avec le texte "Bonjour, TypeScript!"
  • Ajouter le <h1> dans le <div>
  • Ajouter le <div> dans le <body>

✅ Réponse :

Pour accomplir cette tâche, suivez les instructions fournies dans le lien suivant : webdev101.gitlab.io, puis modifiez le fichier src/pages/index.astro comme suit : 


---
---
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Astro + TypeScript</title>
  </head>
  <body>
    <script src="../scripts/app.ts"></script>
  </body>
</html>

Ensuite, créez un dossier src/scripts et ajoutez un fichier nommé app.ts avec le code TypeScript suivant : 


// créer programmatiquement (depuis le script TypeScript) un div avec un id "app"
const appDiv = document.createElement("div");
appDiv.id = "app";

// créer un h1 avec le texte "Bonjour, TypeScript!"
const heading = document.createElement("h1");
heading.textContent = "Bonjour, TypeScript!";

// ajouter le h1 dans le div
appDiv.appendChild(heading);

// ajouter le div dans le body
document.body.appendChild(appDiv);

Ce script ajoutera dynamiquement une structure HTML dans la page une fois qu'elle est chargée, permettant à l'utilisateur de voir le message « Bonjour, TypeScript! » généré par TypeScript.
